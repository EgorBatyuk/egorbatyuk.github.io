НЕОБХОДИМО иметь установленный питон (или найти хост)

в строке, где указывается путь к файлу, необходимо прописать cmd и нажать enter
откроется терминал, где необходимо ввести run_server.bat и нажать enter
выполнется комадпо по активации виртуального окружения
далее необходимо в консоле ввести run.bat и нажать enter 
запустится локальный хост по адрессу http://127.0.0.1:8000/ 
можно пользоваться
